## 📐 Architecture Overview

![SQL Portfolio Architecture](images/architecture.svg)

## 🎯 Skills Demonstrated

- **Window Functions**: ROW_NUMBER, RANK, DENSE_RANK, LAG, LEAD, NTILE
- **Common Table Expressions (CTEs)**: Recursive and non-recursive
- **Complex JOINs**: Self-joins, multiple table joins, conditional joins
- **Aggregations**: GROUP BY with HAVING, ROLLUP, CUBE
- **Subqueries**: Correlated and non-correlated
- **Performance Optimization**: Index strategies, query plans
- **Data Transformation**: PIVOT, UNPIVOT, CASE statements

## 🔍 Query Categories

### 1. Customer Analytics
- Customer Lifetime Value (CLV) calculation
- Cohort analysis and retention metrics
- RFM (Recency, Frequency, Monetary) segmentation
- Churn prediction indicators

### 2. Sales Analysis
- Year-over-year growth comparison
- Moving averages and trend analysis
- Top performers identification
- Sales forecasting queries

### 3. Employee Hierarchy
- Recursive CTEs for org charts
- Manager chain traversal
- Department rollup reports

### 4. Financial Reporting
- Running totals and balances
- Month-end close queries
- Variance analysis

### 5. Fraud Detection
- Anomaly detection patterns
- Transaction velocity checks
- Duplicate detection queries

## 🚀 Getting Started

1. Clone this repository
2. Run `schema/database_schema.sql` to create tables
3. Load `sample_data/sample_data.sql` for test data
4. Execute queries in your preferred SQL client
